Changelog officiel GHI

Cette page présente une timeline lisible des principales évolutions du projet GHI (site, moteur, API et gouvernance). La source de vérité normative reste le fichier CHANGELOG.md du dépôt GitHub.

Gouvernance

vGovernance 0.1 — Cadre de gouvernance (founder-led)

Cette entrée formalise le cadre initial de gouvernance du projet GHI, sans modification fonctionnelle du site, du moteur ou de l’API.

  • Publication du document “GHI — Scope & Governance Principles (v0.1)”.
  • Confirmation d’un modèle piloté par le fondateur, avec la transparence et l’auditabilité comme mécanismes centraux de confiance.
  • Définition explicite du périmètre et du non-périmètre de GHI.
  • Mise en place de la discipline de changement :
    • versionnement sémantique,
    • changelog public,
    • journal de décisions documenté.
  • Clarification des conditions futures de formalisation de la gouvernance (usage externe, financement, arbitrage).

Cette version établit un socle de gouvernance clair et auditable, sans impact sur les indicateurs, données ou visualisations.

GHI Official Changelog

This page provides a readable timeline of major GHI milestones (website, engine, API and governance). The authoritative source of truth remains the repository’s CHANGELOG.md.

Governance

vGovernance 0.1 — Governance framework (founder-led)

This entry formalizes the initial governance framework of the GHI project, without introducing functional changes to the website, engine or API.

  • Publication of “GHI — Scope & Governance Principles (v0.1)”.
  • Confirmation of a founder-led model with transparency and auditability as core trust mechanisms.
  • Explicit definition of scope and non-scope.
  • Introduction of change discipline:
    • semantic versioning,
    • public changelog,
    • documented decision log.
  • Clarification of future governance formalization triggers (external usage, funding, dispute resolution).

This version establishes a clear and auditable governance baseline.

Engine & API

v0.4.2 — Major Sandbox Engine update

This release introduces a more coherent synthetic model, a unified API structure and improved stability. Fully backward-compatible.

Main changes

  • Unified API format with enriched metadata.
  • New endpoints: /metadata, /network, /forecast.
  • Improved synthetic model and harmonised regional costs.
  • CI-compliant deterministic outputs.
  • Updated documentation.

Status

  • Sandbox Engine: active
  • Public API: stable
  • Legacy API: supported
