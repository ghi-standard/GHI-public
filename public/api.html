API GHI v1.0

L’API publique du Global HashCost Index expose un ensemble de points d’accès stables permettant d’obtenir des coûts mondiaux de production du Bitcoin. Toutes les données actuelles proviennent du Sandbox Engine v0.4.0.

Métadonnées moteur (Sandbox Engine v0.4.0)

Toutes les réponses incluent un bloc de métadonnées permettant d’identifier précisément la version de l’API, la méthodologie et le moteur utilisé :

  • api_version : version de l’API publique, ex. v1.0.
  • methodology_version : version de la méthodologie GHI, ex. 1.0.0.
  • engine_version : version du moteur d’exécution, ex. v0.4.0-sandbox.
  • data_origin : origine des données (sandbox pour données factices).
  • stability_seed : entier technique assurant la stabilité du jeu de données.

Base URL

/v1/ghi

1) GET /v1/ghi/indicator

Renvoie l’indicateur quotidien : coût minimum, moyen et maximum (USD) avec métadonnées.

curl "http://127.0.0.1:8000/v1/ghi/indicator"

2) GET /v1/ghi/snapshot

Renvoie un snapshot global structuré au format GHI v1.0.

Endpoints hérités (stubs v1.0, utilisés par la suite de tests)

  • /v1/ghi/history — renvoie une liste d’échantillons historiques.
  • /v1/ghi/regions — renvoie une liste de régions disponibles.
  • /v1/ghi/regions/{region_id} — renvoie les détails d’une région.
  • /v1/ghi/stats — expose des statistiques globales simplifiées.

GHI API v1.0

The Global HashCost Index public API provides stable endpoints giving access to global Bitcoin production costs. All current data is powered by the Sandbox Engine v0.4.0.

Engine Metadata (Sandbox Engine v0.4.0)

Every API response includes a metadata block identifying the exact versions used for API, methodology and engine execution:

  • api_version: public API version, e.g. v1.0.
  • methodology_version: methodology version, e.g. 1.0.0.
  • engine_version: execution engine version, e.g. v0.4.0-sandbox.
  • data_origin: data source (sandbox means simulated data).
  • stability_seed: technical integer ensuring deterministic outputs.

Base URL

/v1/ghi

1) GET /v1/ghi/indicator

Returns daily min / avg / max Bitcoin production cost (USD) with metadata.

2) GET /v1/ghi/snapshot

Returns a structured snapshot following the GHI v1.0 standard.

Legacy endpoints (v1.0 stubs, used by test suite)

  • /v1/ghi/history — returns a sample historical list.
  • /v1/ghi/regions — returns the list of available regions.
  • /v1/ghi/regions/{region_id} — returns a region detail.
  • /v1/ghi/stats — exposes simplified global statistics.
