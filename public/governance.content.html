Gouvernance du standard GHI

Le Global HashCost Index est un standard ouvert. Sa gouvernance repose sur la transparence, le versionnage contrôlé et la publication systématique des évolutions.

Principes

  • L’API, la méthodologie et le moteur possèdent chacun leur version indépendante.
  • Tout changement majeur entraîne une nouvelle version majeure.
  • Toutes les modifications passent par des Pull Requests publiques sur GitHub.

Processus officiel

  • Proposition : création d’une issue ou d’une Pull Request GitHub.
  • Revue : analyse technique, méthodologique et d’intégration.
  • Validation : merge et création d’une release officielle.

GHI Governance

The Global HashCost Index is an open standard. Governance relies on transparency, controlled versioning and a fully public development workflow.

Principles

  • The API, methodology and engine each maintain their own version.
  • Breaking changes require a new major version.
  • All modifications go through public GitHub Pull Requests.

Official process

  • Proposal: GitHub issue or Pull Request.
  • Review: technical, methodological and integration checks.
  • Validation: merge and official release creation.
