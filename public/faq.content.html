FAQ – Global HashCost Index

1. Le GHI prédit-il le prix du Bitcoin ?

Non. Le GHI mesure un coût de production estimé, pas un prix futur. Il décrit la structure de coûts du réseau, pas sa valorisation.

2. Les données actuelles sont-elles réelles ?

La version publique utilise un moteur Sandbox : les valeurs chiffrées sont de démonstration, mais les schémas d’API et la méthodologie sont ceux de la version v1.0.

3. Puis-je utiliser GHI dans un rapport institutionnel ?

Oui, à condition de mentionner la version utilisée (API v1.0, Méthodo v1.0.0, Engine v0.3.0) et la source : “Global HashCost Index (GHI) – documentation officielle”.

4. Comment accéder à l’API ?

Voir la page API et la documentation complète sur GitHub.

FAQ – Global HashCost Index

1. Does GHI predict Bitcoin’s price?

No. GHI measures estimated production costs, not future prices. It is an economic indicator of the network, not a pricing model.

2. Are current data points real?

The public version uses a Sandbox engine: numbers are demo values, while the API schemas and methodology correspond to v1.0.

3. Can GHI be used in institutional reports?

Yes, as long as the versions used are clearly indicated and the source is cited as the official GHI documentation.
