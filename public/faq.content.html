FAQ – Global HashCost Index

1. Le GHI prédit-il le prix du Bitcoin ?

Non. Le GHI mesure un coût de production estimé, pas un prix futur. Il décrit la structure économique du réseau, pas sa valorisation de marché.

2. Les données affichées sont-elles réelles ?

La version publique repose sur un moteur Sandbox. Les valeurs chiffrées sont illustratives, mais les structures d’API et la méthodologie correspondent au standard officiel GHI.

3. Puis-je utiliser GHI dans un rapport institutionnel ?

Oui. Il est recommandé de mentionner explicitement la version de l’API, de la méthodologie et du moteur utilisées, ainsi que la source : Global HashCost Index (GHI) – documentation officielle.

4. Comment accéder à l’API ?

Consulte la page API et le Quickstart, ou la documentation complète disponible sur GitHub.

FAQ – Global HashCost Index

1. Does GHI predict Bitcoin’s price?

No. GHI estimates production costs, not future prices. It is an economic indicator of the network, not a pricing model.

2. Are the displayed data points real?

The public version relies on a Sandbox engine. Numerical values are illustrative, while the API structures and methodology follow the official GHI standard.

3. Can GHI be used in institutional reports?

Yes. It is recommended to explicitly state the API, methodology and engine versions used, and to cite the source as the official GHI documentation.

4. How can I access the API?

See the API page and the API Quickstart, or consult the full documentation on GitHub.
