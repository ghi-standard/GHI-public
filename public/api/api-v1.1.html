GHI Public API v1.1

API synthétique et entièrement documentée – version 1.1 (FR/EN). Elle reste compatible avec les usages v1.0, tout en introduisant un format de réponse unifié et de nouveaux endpoints.

1. Aperçu (FR)

L’API v1.1 améliore la structure du snapshot, ajoute des statistiques globales et introduit trois nouveaux endpoints : /metadata, /network et /forecast. Toutes les données proviennent du Sandbox Engine v0.4.0.

Principales améliorations

  • Métadonnées unifiées dans toutes les réponses : version, timestamp_utc, engine_sandbox_version.
  • Statistiques globales de coût (min / avg / max).
  • Structure normalisée des régions.
  • Nouveaux endpoints de métadonnées et de réseau.

2. Endpoints

Endpoint Description
/v1/ghi/snapshot Coûts de production régionaux (min/avg/max) + statistiques globales.
/v1/ghi/history Série temporelle synthétique des coûts (min/avg/max).
/v1/ghi/regions Liste des régions disponibles (id + nom).
/v1/ghi/metadata Métadonnées globales de l’API et du moteur.
/v1/ghi/network Métriques réseau synthétiques (hashrate, difficulty, avg fee).
/v1/ghi/forecast Prévision synthétique du coût moyen de production.

3. Structure de réponse – exemple de snapshot

{
  "version": "1.1",
  "timestamp_utc": "2025-12-09T12:00:00Z",
  "engine_sandbox_version": "0.4.0-sandbox",
  "region_count": 3,
  "regions": [
    {
      "region_id": "us",
      "currency": "USD",
      "min_cost_usd": 38000.0,
      "avg_cost_usd": 42000.0,
      "max_cost_usd": 50000.0
    }
  ],
  "global_min_cost_usd": 35000.0,
  "global_avg_cost_usd": 42000.0,
  "global_max_cost_usd": 52000.0
}

4. Métadonnées communes

Toutes les réponses v1.1 partagent les mêmes champs de métadonnées :

  • version – version de l’API ("1.1").
  • timestamp_utc – horodatage ISO 8601 UTC.
  • engine_sandbox_version – ex. "0.4.0-sandbox".

GHI Public API v1.1

Open, synthetic and fully documented API – version 1.1 (EN/FR). It keeps v1.0 compatibility while introducing a unified response format and new endpoints.

1. Overview (EN)

API v1.1 improves the snapshot structure, adds global cost statistics and introduces three new endpoints: /metadata, /network and /forecast. All data is powered by the Sandbox Engine v0.4.0.

Main improvements

  • Unified metadata in every response: version, timestamp_utc, engine_sandbox_version.
  • Global min / avg / max production cost.
  • Normalized regional structure.
  • New metadata and network endpoints.

2. Endpoints

Endpoint Description
/v1/ghi/snapshot Synthetic regional production costs (min/avg/max) + global stats.
/v1/ghi/history Synthetic historical cost series (min/avg/max).
/v1/ghi/regions List of available regions (id + name).
/v1/ghi/metadata Global API and engine metadata.
/v1/ghi/network Synthetic network metrics (hashrate, difficulty, avg fee).
/v1/ghi/forecast Synthetic forecast of the average production cost.

3. Snapshot response structure (example)

  "global_min_cost_usd": 35000.0,
  "global_avg_cost_usd": 42000.0,
  "global_max_cost_usd": 52000.0

4. Common metadata

Every v1.1 response shares the same metadata fields:

  • version – API version ("1.1").
  • timestamp_utc – ISO 8601 UTC timestamp.
  • engine_sandbox_version – e.g. "0.4.0-sandbox".
