GHI Public API v1.1

Open, synthetic and fully documented API – version 1.1 (EN/FR).

1. Overview (EN)

API v1.1 introduces a unified response format, improved snapshot structure, global statistics, and three new endpoints: /metadata, /network, and /forecast. It remains fully compatible with the GHI v1.0 public standard.

Main improvements:

  • Unified metadata in all responses: version, timestamp_utc, engine_sandbox_version
  • Global cost statistics (min/avg/max)
  • Normalized regional structure
  • New endpoints

2. Endpoints

EndpointDescription
/v1/ghi/snapshotSynthetic regional production costs
/v1/ghi/historyHistorical cost values
/v1/ghi/regionsList of regions
/v1/ghi/metadataAvailable endpoints
/v1/ghi/networkBitcoin network synthetic metrics
/v1/ghi/forecastCost forecast

L’API v1.1 améliore la normalisation des réponses, introduit des statistiques globales et ajoute trois nouveaux endpoints (/metadata, /network, /forecast). Elle reste conforme au standard public GHI v1.0.

Principales évolutions :

  • Métadonnées unifiées
  • Statistiques globales consolidées

  • Nouveaux endpoints
