À propos du Global HashCost Index

Le Global HashCost Index (GHI) est un standard ouvert conçu pour mesurer le coût mondial de production du Bitcoin. Il vise à fournir un indicateur neutre, transparent et reproductible du réseau.

Principes

  • Ouvert : documentation et code accessibles publiquement.
  • Versionné : chaque évolution majeure fait l’objet d’une publication formelle.
  • Neutre : aucune optimisation pour un acteur ou une région spécifique.
  • Vérifiable : implémentation de référence disponible.

About the Global HashCost Index

The Global HashCost Index (GHI) is an open standard designed to measure the global cost of producing Bitcoin. It provides a neutral, transparent and reproducible indicator of network production costs.

Principles

  • Open: public documentation and source code.
  • Versioned: major evolutions are formally published.
  • Neutral: no optimisation for any specific actor or region.
  • Verifiable: reference implementation available.
