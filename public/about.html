À propos du Global HashCost Index

Le Global HashCost Index (GHI) est un standard ouvert conçu pour mesurer le coût mondial de production du Bitcoin. Il vise à fournir un indicateur neutre, transparent et reproductible du réseau.

Principes

  • Ouvert : documentation et code accessibles publiquement.
  • Versionné : chaque évolution importante est liée à une version.
  • Neutre : aucune optimisation pour un acteur particulier.
  • Vérifiable : implémentation de référence disponible.

About the Global HashCost Index

The Global HashCost Index (GHI) is an open standard designed to measure the global cost of producing Bitcoin. It provides a neutral, transparent and reproducible indicator of the network’s production costs.
