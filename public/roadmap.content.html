Feuille de route GHI – Roadmap v1.0

Cette feuille de route présente les prochaines étapes du Global HashCost Index, organisées par version et par niveau de maturité. Le standard GHI évolue selon trois axes : méthodologie, moteur et infrastructure API.

1. Version publique actuelle – v1.0.0

  • Méthodologie v1.0.0 publiée (FR/EN).
  • Sandbox Engine v0.4.0 opérationnel.
  • API publique v1.0 opérationnelle.
  • Documentation complète : Quickstart, Méthodologie, Transparence, Institutions.
  • Architecture stable pour intégration tiers.

2. Travaux en cours – v1.1 à v1.x

Objectif : compléter le standard GHI v1.x avant le passage à v2.0.

  • Ajout des pages Régions (Amérique du Nord, Europe, Asie hors Chine, Reste du monde).
  • Affinement des profils régionaux (machine mix, tarifs synthétiques).
  • Mise à jour progressive de la documentation API.
  • Stabilisation du moteur sandbox (v0.5.0).
  • Support amélioré pour dashboards (export, CSV/JSON structurés).

3. Version majeure – v2.0 (prochain grand chantier)

La version 2.0 représente la transition vers un moteur réel (non-sandbox) avec données consolidées et gouvernance renforcée.

  • Modèle méthodologique v2.0 (ajustements, scénarios étendus).
  • Data Model v2.0 (structures régionales plus détaillées).
  • Intégration de données multi-sources vérifiables.
  • Moteur GHI Engine v2.0 (calcul réel, non synthétique).
  • Profil énergétique horaire par région.
  • Segments par type d’opérateur (industriel, retail).
  • Metrics CO₂, énergie primaire, mix énergétique.
  • Refonte API → API v2.0.

4. Vision long terme (3.x – 5.x)

  • GHI en temps réel (rolling snapshots).
  • Réseau de contributeurs régionaux certifiés.
  • Indices sectoriels (ASIC efficiency index, Grid Mix Index, etc.).
  • Modèle open data (export complet, licences publiques).

GHI Roadmap – v1.0

This roadmap outlines the upcoming milestones of the Global HashCost Index, organised by version and maturity level. GHI evolves along three axes: methodology, engine, and API infrastructure.

1. Current public version – v1.0.0

  • Methodology v1.0.0 published (FR/EN).
  • Sandbox Engine v0.4.0 operational.
  • Public API v1.0 available.
  • Complete documentation: Quickstart, Methodology, Transparency, Institutions.
  • Stable architecture for integrations.

2. Work in progress – v1.1 to v1.x

Goal: complete the GHI v1.x standard before transitioning to v2.0.

  • Creation of Region pages (North America, Europe, Asia ex-China, Rest of World).
  • Refinement of synthetic regional profiles.
  • Progressive update of API documentation.
  • Sandbox engine improvements (v0.5.0).
  • Enhanced dashboard and export support (CSV/JSON).

3. Major release – v2.0 (next large milestone)

Version 2.0 represents the transition to a real engine with consolidated data and enhanced governance.

  • Methodology v2.0 (revised assumptions, advanced scenarios).
  • Data Model v2.0 (richer regional structures).
  • Integration of verifiable multi-source datasets.
  • GHI Engine v2.0 (real computation, non-synthetic).
  • Hourly electricity profiles per region.
  • Segmentation by miner type (industrial vs retail).
  • CO₂ and primary energy metrics.
  • API v2.0 redesign.

4. Long-term vision (v3.x – v5.x)

  • Real-time GHI (rolling snapshots).
  • Certified regional contributor network.
  • Sector sub-indices (ASIC efficiency index, Grid Mix Index, etc.).
  • Open data model with full public export.
