# GHI Version Freeze Declaration

This document formally declares the frozen versions of the
Global HashCost Index public standard.

## Frozen versions

- GHI Public Standard: v1.1.0
- GHI API: v1.1.0
- GHI Sandbox Engine: v0.4.3-sandbox

## Freeze policy

Frozen versions:
- MUST NOT be modified
- MUST remain backward compatible
- MUST be referenced by immutable Git tags

Any change requires a new semantic version.

## Authority

This declaration is normative and applies to all public repositories
under the ghi-standard organization.
