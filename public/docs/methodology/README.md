# Méthodologie — GHI (Global HashCost Index)

Ce dossier contient les **documents méthodologiques sources** du projet **GHI – Global HashCost Index**.

Ces fichiers constituent la **référence officielle** décrivant le modèle, ses hypothèses, ses formules
et ses principes de calcul.

---

## 1. Statut des documents

Tous les documents présents dans ce dossier sont considérés comme :

> **Sources de vérité méthodologiques**

Ils sont :
- rédigés en **Markdown (`.md`)**,
- versionnés dans Git,
- auditables,
- destinés à être lus, critiqués et reproduits.

Les formats dérivés (PDF, HTML) sont générés à partir de ces sources
et ne doivent jamais être modifiés directement.

---

## 3. Règles de modification

Autorisé
	•	Modifier ou améliorer les fichiers .md
	•	Ajouter des précisions méthodologiques documentées
	•	Mettre à jour les hypothèses ou formules avec justification

Interdit
	•	Modifier un PDF sans modifier la source Markdown correspondante
	•	Dupliquer un même contenu méthodologique dans plusieurs fichiers sans lien explicite
	•	Introduire un document sans statut clair (source / complément)

⸻

## 4. Génération des formats dérivés

Les fichiers Markdown de ce dossier peuvent être utilisés pour générer :
	•	des documents PDF institutionnels,
	•	des pages HTML publiques,
	•	des supports d’audit ou de revue externe.

Ces formats sont considérés comme artefacts de publication.

⸻

## 5. Objectif de gouvernance

Cette organisation vise à garantir :
	•	la transparence méthodologique,
	•	la reproductibilité du modèle,
	•	la traçabilité des évolutions,
	•	la conformité aux attentes institutionnelles et d’audit.
