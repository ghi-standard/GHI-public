v0.1.0 – Prototype interne

  • Premiers calculs de coût global de production du Bitcoin (ordre de grandeur).
  • Tests exploratoires sur la faisabilité d’un indice ouvert et documenté.

GHI Changelog

v0.4.1 – Public site refresh

  • Unified design across all public pages (shared layout, dark theme).
  • Bilingual navigation FR / EN generated from .content.html files.
  • Improved Dashboard (pre-release) page with clearer example data.
  • Release workflow prepared (feature / release branches, mandatory PRs into main).

v0.4.0 – Semi-operational sandbox

  • Added a “fake data” sandbox API mirroring the future GHI production model.
  • Dedicated API documentation page (“GHI API – Sandbox”).
  • Structured local and remote endpoints for snapshots and historical data.

v0.3.0 – Institutional public site

  • Launch of globalhashcostindex.org (introductory public version).
  • Dedicated pages: Methodology, Regions, Governance, GHI for Institutions.
  • Base layer of public documentation for early discussions with institutional actors.

v0.2.0 – GHI model (internal)

  • Definition of GHI Data Model v1.0 (global and regional cost structures).
  • First Python scripts to aggregate and normalise data.

v0.1.0 – Internal prototype

  • First computations of the global Bitcoin production cost (order of magnitude only).
  • Exploratory tests on the feasibility of an open, documented index.
