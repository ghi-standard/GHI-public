v0.1.0 — Première publication publique

30 novembre 2025

Résumé

Première publication officielle du Global HashCost Index (GHI). Publication du standard, de la documentation institutionnelle et du site GHI.

Ajouts principaux

  • Whitepaper FR et EN.
  • Documents « Méthodologie & Transparence » FR et EN.
  • GHI pour les Institutions FR et EN.
  • Standard technique GHI Technical Standard v1.0.
  • Modèle de données GHI-Data-Model-v1.0.
  • Site GHI-public (overview, méthodologie, régions, API, roadmap, gouvernance).
  • Référence API (sandbox) : API_REFERENCE.md.

Notes

Aucun moteur réel n’est exposé à ce stade. Cette release fige l’antériorité du standard et prépare la Phase 2.

v1.0.0 — Moteur de production

Phase 3 — planifiée

Objectif

Première version institutionnelle du moteur GHI réel : calculs min/moy/max basés sur le hashrate réel, l’énergie, le mix machines, par région.

Contenu prévu

  • Moteur GHI complet (calcul réel).
  • Snapshots quotidiens, puis intrajournaliers.
  • Historique complet.
  • API de production sécurisée, versionnée, stable.
  • Exports institutionnels (CSV / Parquet).

Institutionnel

Documentation renforcée, versioning strict, gouvernance du standard, compatibilité ESG et extension CO₂.

v0.1.0 — Initial public release

30 November 2025

Summary

First official publication of the Global HashCost Index (GHI). Release of the standard, institutional documentation and the GHI website.

Main additions

  • Whitepaper in French and English.
  • “Methodology & Transparency” documents in French and English.
  • “GHI for Institutions” documents in French and English.
  • Technical standard GHI Technical Standard v1.0.
  • Data model GHI-Data-Model-v1.0.
  • GHI-public website (overview, methodology, regions, API, roadmap, governance).
  • API Reference (sandbox): API_REFERENCE.md.

Notes

No real engine is exposed at this stage. This release fixes the priority date of the standard and prepares Phase 2.

v0.2.0 — Sandbox engine & API

Phase 2 — upcoming

Overview

Deployment of the sandbox engine ghi-engine and the structured FastAPI backend, using synthetic data that are fully aligned with the GHI v1.0 model.

    • /v1/ghi/history
    • /v1/ghi/regions
    • /v1/ghi/regions/{region_id}
    • /v1/ghi/stats
  • Mocked engine (FakeEngine) fully aligned with the GHI Data Model.
  • Automated tests (Pytest).
  • Updated API pages and public documentation.

v1.0.0 — Production engine

Phase 3 — planned

Goal

First institutional version of the real GHI engine: min/avg/max cost based on real hashrate, energy, machine mix, by region.

Planned content

  • Full GHI engine (real calculations).
  • Daily, then intraday snapshots.
  • Full history.
  • Secure, versioned, stable production API.
  • Institutional exports (CSV / Parquet).

Institutional

Strengthened documentation, strict versioning, standard governance, ESG compatibility and CO₂ extension.
