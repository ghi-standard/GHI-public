Résumé

La méthodologie GHI v1.0 définit une façon standardisée, transparente et reproductible de mesurer le coût mondial de production du Bitcoin.

  • Coûts exprimés en USD par BTC
  • Trois métriques : min / avg / max cost
  • Indicateur global journalier
  • Snapshot global (structure longue, multi-jour)
  • Moteur Sandbox utilisé pour stabiliser l’API

Principes

  • Standard de transparence : versionning, API stable
  • Non prédictif : mesure les coûts, pas le prix futur
  • Conçu pour les institutions, analystes & régulateurs
  • Le moteur réel (données mondiales) suivra sans casser l’API
