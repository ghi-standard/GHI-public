# GHI Community Code of Conduct

The Global HashCost Index (GHI) is an open scientific standard.  
We are committed to fostering an inclusive, respectful, and collaborative environment.

This Code of Conduct applies to:
- maintainers
- contributors
- institutional partners
- researchers using the standard

---

## 1. Standards of Behavior

Participants must:

- Show respect in all interactions.
- Debate ideas, not individuals.
- Provide constructive feedback.
- Accept technical decisions made through governance.

Unacceptable behavior includes:

- Harassment or discrimination.
- Personal attacks.
- Disruptive or aggressive communication.
- Sharing confidential or private information.

---

## 2. Reporting

If you observe a violation, please report it to:

**ghi-standard@protonmail.ch**  
This inbox is monitored by the governance committee.

Reports are treated confidentially.

---

## 3. Enforcement

Violations may result in:

1. Private warning  
2. Temporary interaction limits  
3. Ban from contribution activity  
4. Escalation to institutional partners (if relevant)

Decisions are final and at the discretion of the committee.

---

## 4. Acknowledgment

This Code of Conduct is inspired by established open-standard communities and adapted specifically for the GHI governance model.
