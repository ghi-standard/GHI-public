# Dashboard GHI

Le dashboard propose une visualisation intuitive du modèle :

- Mise en page inspirée d’Apple
- Tailwind CSS (CDN)
- Chart.js (CDN)
- Données en direct via API v1

---

## Sections

### 1. Vue globale
- Prix du BTC
- Coût moyen global
- Distance au coût
- Bande min–max

### 2. Graphique historique
- Prix BTC vs coût moyen GHI

### 3. Régions
Cartes individuelles :
- nom
- part du hashrate
- coût moyen
- min / max
