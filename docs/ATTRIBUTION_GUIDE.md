# GHI Attribution Guide

This document explains how to correctly reference the GHI standard in  
papers, dashboards, websites or software products.

---

## 1. Short Attribution (default)

Use this format when referencing GHI values or documentation:

**EN**  
“Data based on the Global HashCost Index (GHI), an open standard maintained by the GHI Standard group.”

**FR**  
“Données basées sur le Global HashCost Index (GHI), un standard ouvert maintenu par le groupe GHI Standard.”

---

## 2. Long Attribution (academic)
Global HashCost Index (GHI), GHI Standard Group.
Public methodology and specifications available at: https://ghi-standard.org

---

## 3. For dashboards or websites

You must display:

- GHI name  
- A clickable link to GHI-public  
- The date/time of the snapshot used  

Example:

> Powered by Global HashCost Index — https://github.com/ghi-standard/GHI-public

---

## 4. For code reuse

If you reuse fragments of documentation or pseudocode:

Include:
Source: GHI Public Standard (MIT License)
https://github.com/ghi-standard/GHI-public
