# GHI – Global HashCost Index

The Global HashCost Index (GHI) is an open, transparent and reproducible standard that measures the global cost of Bitcoin production.  
Built on a structured data model, GHI provides methodology-based estimates of minimum, average and maximum production costs across major world regions, weighted by real hashrate distribution.

GHI is designed for:
• researchers and analysts  
• institutions and regulators  
• miners and infrastructure providers  
• macro and financial actors  

The goal: provide a clear, neutral and documented indicator of Bitcoin production costs globally.

GHI (Global HashCost Index) est un standard destiné à mesurer et suivre le coût de production du Bitcoin à l’échelle mondiale, en combinant :

- les données énergétiques et régionales,
- les caractéristiques des machines de minage,
- les profils d’efficacité selon les zones géographiques,
- des indicateurs synthétiques de stress économique pour les mineurs.

Ce dépôt public publie la **vision, la méthodologie générale et les documents de standardisation** de GHI, sans exposer le moteur de calcul interne ni l’implémentation technique complète.

---

## Objectifs

- Définir un référentiel indépendant pour le coût du hashrate Bitcoin.
- Offrir un outil exploitable par les institutions, analystes, gestionnaires d’actifs et chercheurs.
- Améliorer la transparence économique et énergétique du minage Bitcoin.

## Contenu de ce dépôt

Ce repo contient :

- La documentation générale :  
  - Whitepaper (version publique)  
  - Documents “GHI pour les institutions”  
  - Dossier “Méthodologie + Transparence”  
  - Documents de gouvernance  
  - Roadmap publique  
  - GHI Technical Standard (v1.0)

- Les pages de présentation :  
  - Pages HTML d’overview, dashboard, méthodologie, régions, roadmap, etc.  
  - Feuilles de style associées (CSS).

- Le schéma d’API au niveau conceptuel :  
  - Description de ce que l’API GHI expose à terme (réponses-types, structure de données),  
  - sans endpoints actifs ni accès au moteur réel.

Aucun code sensible n’est publié dans ce dépôt :   
le moteur de calcul, les scripts internes, la configuration détaillée et l’historique opérationnel restent privés.

---

## Ce que ce dépôt ne contient pas

- Aucune implémentation du moteur GHI (Python ou autre).
- Aucune formule détaillée propriétaire.
- Aucune configuration interne de production.
- Aucune donnée de production en temps réel.

Ces éléments sont conservés dans des dépôts privés ou des environnements séparés.
